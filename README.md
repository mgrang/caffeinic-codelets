# caffeinic-codelets
## Coffee, Code, Repeat

Implementations of various algorithms and data structures in C++.

### Usage
Initialize a new C++ program:
```
./init foobar.cpp

cat foobar.cpp

#include <iostream>
using namespace std;

int main() {
}
```

Compile and run the program:
```
./bld foobar.cpp
```
